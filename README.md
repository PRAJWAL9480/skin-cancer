# Melanoma Spotter

Melanoma Spotter is a deep learning based web application
for automated skin cancer detection using an ensemble
of VGG16 and DenseNet121 models.

## Features
- Skin lesion validation using CNN
- Hybrid ensemble model
- Web-based image upload
- Confidence-based prediction

## Architecture
User → Web UI → Flask API → Skin Detector → Ensemble Model → Result

## Technologies Used
- Python
- TensorFlow / Keras
- Flask
- HTML, CSS, JavaScript

## Dataset
- HAM10000 Skin Cancer Dataset

## How to Run
1. Install dependencies
2. Run Flask server
3. Upload image via browser
